# About The Project

This is a rest api(crud app).

## Built With

- [Express](https://expressjs.com/)
- [MongoDb](https://www.mongodb.com/)
- [TypeScript](https://www.typescriptlang.org/)

### Prerequisites

- npm

  ```in root dir
        npm run build
        npm run dev
  ```

- yarn

  ```in root dir
        yarn build
        yarn dev
  ```
